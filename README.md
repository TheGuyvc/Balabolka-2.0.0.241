## 📖 About

Balabolka 2.0.0.241 is a versatile text-to-speech utility for Windows and macOS, designed to read aloud any text from files, clipboard, or typed input. It leverages multiple SAPI voices to deliver clear, customizable speech output, and supports saving audio to MP3, WAV, or OGG formats. Whether you need to listen to documents, proofread, or create audio files, Balabolka provides a reliable and efficient solution.

## ⚙️ Requirements

- **Operating System**: Windows 10/11 (64-bit) or macOS 10.15+
- ** Disk Space**: At least 150 MB for installation and voice data
- **Internet Connection**: Required for initial  only; no online activation needed
- **Runtime**: None required; self-contained executable
- **Audio Output**: Standard sound card or integrated audio

## ✨ Features

**Text-to-Speech Engine** 🎤 — Reads text from documents, web pages (via clipboard), or direct input with natural-sounding voices.

**Multiple Voice Support** 🗣️ — Supports SAPI5 and Microsoft Speech Platform voices, plus optional third-party voices.

**Audio File Export** 💾 — Save speech as MP3, WAV, OGG, or WMA files for offline listening or podcast creation.

**Batch Processing** 📂 — Convert entire folders of text documents (TXT, DOCX, PDF) into audio files automatically.

**Customizable Pronunciation** 🔧 — Fine-tune word pronunciation and speaking rate with a built-in dictionary editor.

**Subtitle & Bookmark Sync** 📑 — Save time-aligned subtitles (SRT) and bookmarks for long audio files or audiobooks.

**System Tray Integration** 🔔 — Minimize to tray for quick access and background playback.

**Multi-language Interface** 🌐 — Full support for over 20 languages in the user interface.

## 🔧 Configuration

Balabolka can be fine-tuned via its interface or through an optional configuration file (`balabolka.ini`) placed in the same directory as the executable.

**Example `balabolka.ini` snippet:**
```ini
[Main]
Voice=Microsoft David Desktop
Rate=0
Volume=100
OutputFormat=mp3
SubtitleFormat=srt
```

*Note: The application automatically reads this file on startup if present; all settings are adjustable within the GUI.*

## ❓ FAQ

**Is there any risk of detection or ban when using Balabolka?**  
Balabolka is a legitimate text-to-speech utility; it does not interact with online games or anti- systems. There is no risk of bans or detection from any platform.

**The audio output is choppy or stuttering — how can I fix it?**  
Ensure your audio drivers are up to date. In Balabolka's settings, reduce the voice speed or switch to a different SAPI voice. Also close other applications that may be using audio resources.

**Can I use custom voices with Balabolka?**  
Yes. You can install any SAPI5-compatible voice (e.g., from Microsoft or third-party vendors). After installation, the new voice will appear in the voice selection dropdown.

**How do I convert a PDF to audio?**  
Open the PDF file using Balabolka's "Open File" dialog. The tool will extract the text and allow you to play or export it as an audio file.
